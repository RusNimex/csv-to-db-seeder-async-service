# Сервис асинхронной загрузки CSV-файлов

Сервис для асинхронной загрузки и обработки CSV-файлов с данными о компаниях, городах и категориях.

## Архитектура

Сервис состоит из двух основных компонентов:

### API сервис (PHP)
- Принимает CSV-файлы через HTTP API
- Сохраняет файлы в хранилище
- Публикует задачи на обработку в RabbitMQ с приоритетами (high, normal, large)

### Worker сервис (Go)
- Обрабатывает задачи из очередей RabbitMQ
- Парсит CSV-файлы с разделителем `;`
- Сохраняет данные в MySQL базу данных
- Поддерживает обработку нескольких очередей параллельно

## Обрабатываемые данные

CSV-файлы должны содержать следующие поля:
- Название компании
- Регион
- Район
- Город
- Email
- Телефон
- Рубрика (категория)
- Подрубрика (подкатегория)

## Технологии

- **API**: PHP 8.x, RabbitMQ
- **Worker**: Go, RabbitMQ, MySQL
- **Очереди**: RabbitMQ с приоритетами и разделением по размеру файлов
